एचटीएम्एल ५(5) और सीएसएस ३(3) वेब फोंट्स का उदाहरण

WORKS ONLY IN OPERA 10 AND SAFARI 4 SO FAR.

Notice the difference in the spacing, the slants and the curves of the letters.

The different hindi fonts are achieved using the @font-face rule introduced in CSS3. The @font-face rule allows you to use any custom or external fonts in your web pages, so that you are not limited to only the few fonts provided by your system by default.
